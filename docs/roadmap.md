## Phase 2: Persistence & Export (Next Steps)
*   **PDF Export**: Implement functionality to download the Visual Dashboard and the Action Plan as a formatted PDF for academic submission.
*   **Local Storage**: Save progress locally so users don't lose data on refresh.
*   **History**: Allow users to compare current audits with previous sessions to track growth.

## Phase 3: Enhanced Gamification
*   **Skill Badges**: Award badges for reaching "Proficient" levels in JISC or Soft Skills.
*   **Avatar Generation**: Use Gemini Image generation to create a professional avatar based on the user's description.
*   **Experience Points (XP)**: Assign XP values to completing sections to visualize progress.

## Phase 4: Integration & personalization
*   **Job Market Data**: Integrate with a Jobs API to pull real-time skills requirements for the specific `careerGoal`.
*   **Learning Resources**: Instead of generic resource suggestions, link to specific courses (LinkedIn Learning, Coursera) or internal University modules.
*   **Authentication**: User accounts to store long-term career development data.

## Phase 5: Institutional Features
*   **Tutor View**: Dashboard for tutors to view aggregated class data (anonymized) to identify common skill gaps.
*   **Peer Review**: Mechanism for students to share their PDP with peers for feedback within the app.
